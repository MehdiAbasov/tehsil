HTML Nədir?

HTML, web səhifələri yaradılarkən işlədilən işarətləmə dildir. İngilizcəsi "hyper-text markup language", Azərbaycancası "hiper mətin işarətləmə dilinin" qısaltmasıdır.

Bir web səhifəsinə girilən zaman, görülən HTML elementləri ilə yazılmışdır. Onların daha gözəl və nizamlı gönünməsi üçün CSS, daha dinamik olması üçün isə Javascriptdən istifadə olunur

CSSvə Javascript dərslərinə HTMLdən sonra baxılması bu sahəyə yeni başlayanlar üçün daha yaxşı olacaqdır. HTMLi öyrəndikdən sonra sıra ilə CSS və Javascript, front-end öyrənmək istəyən insanların getməsi gərəkən bir yoldur

  • HTML, uzun adı ilə "Hyper Text Markup Language" deməktir.
  • HTML, Azərbaycan dilində "Hiper mətin işarətləmə dili" deməktir.
  • HTML ilə yazılar işarətlənərək, web səyfələrinin strukturları müəyyənləşdirilər.
  • HTML ilə işləyərəkən müəyyən olunmuş HTML elementlərindən istifadə olunur.

Sadə Html Səhifəsi

Sadə bir HTML səyfəsi, təməl olaraq aşağıdakı etiketlƏrdən ibarətdir.

<!DOCTYPE html>
<html>
<head>
<title>Başlık</title>
<meta charset="UTF-8">
</head>
<body>

test

</body> 
</html>

Səhifənin Açıqlaması

  • <!doctype>sənədin növünü əks etdirir və brauzerlərə veb səhifələri düzgün göstərməyə kömək edir.
  • <html>kök elementdir və bütün HTML sənədini müəyyən edir .
  • <head>etiketdə səhifə haqqında başlıqlar var. Başqa sözlə, istifadəçinin veb-səhifəyə daxil olanda görə bilmədiyi, ancaq arxa planda yüklənərək əməliyyatı davam etdirəcəyi məlumatları əhatə edir. Bunların ilk nümunəsi olan <title>etiketi yazaraq səhifənin başlığını təyin etdik . Bu etiketin təfərrüatlarına daha sonra gələcəyik.
  • <body>Element sənədin gövdəsini müəyyənləşdirir. HTML-də yazacağımız hər şey <body>teqinin içərisində olur.
  • <title>teqi səhifənin başlığını müəyyən etmək üçün istifadə olunur.
  • <meta>teqi səhifə haqqında metadatanı müəyyən edir. Metadata verilənlər haqqında məlumatdır. <meta>teqi içərisində yazılanlar səhifədə göstərilməyəcək, lakin

Əgər yuxardakı misalı anlamadınızsa tag strukturuna baxın!

HTML-də Tag Strukturu

  • Teqlər HTML-də mətni qeyd etmək üçün istifadə olunur. Və teqləri təyin edərkən <kiçikdir teq böyüktür> işarələrin içərisində yazılır. Yəni <teq> kimi yazılır.
  • Bundan əlavə, açılmış teqlər əvvəlinə /işarəsi qoyularaq bağlanır. yəni <teq> </teg>kimi olur. Lakin bütün teqlər açılıb bağlanmır, bəzi teqlər avtomatik bağlanır. Ona görədə onları bağlamağa ehtiyac yoxdur. Bunların fərqini növbəti dərslərdə Görəcəksiniz. Misal çəkməli olsaq <img> , <br> , <hr>
